Mathematical Equations and Chemical Formulas

Superscript and Subscript Elements

The superscript element is used to display a piece of text as a superscript (<sup>). A superscript is a symbol or letter printed above the normal line of text.

Superscript example 
<p>2<sup>2</sup>(2 squared) is 4.</p>

22 (2 squared) is 4.

Common use cases for the superscript element would include exponents, superior lettering, and ordinal numbers.

Here is an example using the superscript element for superior lettering 
<p>Monseigneur is often written as <strong>M<sup>gr</sup></strong>.</p>
Monseigneur is often written as Mgr.

Superior lettering refers to letters written in superscript, usually to indicate abbreviations. The letters g and r are wrapped inside superscript tags to illustrate the abbreviation in this example.

It is important to note that the superscript element should only be used for typographical reasons.
Note: If you want to style a piece of text with a raised baseline, then you should use CSS instead of the superscript element.

To represent chemical equations inside HTML, you would use the subscript (<sub>) element. This element uses a subscript which displays a lowered baseline using smaller text.

Subscript example

CO2

<p>CO<sub>2</sub></p>

How Do You Represent Computer Code in HTML?

The <code> Element

The inline <code> element is used to represent short snippets of code inside text. Common use cases for the <code> element would be for technical articles and documentation pages.

Code element example 
<p>To create a hyperlink, use the <code><a href="url">link text</a></code> element.</p>

To create a hyperlink, use the <a href="url">link text</a> element.

*By wrapping this code snippet inside <code> tags, it communicates to the browser that the text is a piece of inline code.

If you want to represent multiple lines of code, you will need to place a code element inside a preformatted text element or <pre>.

Preformatted text and code element example 

    body {
        color: red;
    }
*When using the pre element, you will need to be mindful of spacing because it will display exactly as written inside the HTML document.

What Are the U, S, and Ruby Elements Used For, and How Do They Work?

The <u> Element

The unarticulated annotation element, or <u> element for short, is used to represent inline text that has non-textual annotation applied.

<u> element example 
<p>This is an <u>underlined</u> word.</p>

This is an underlined word.

The default styling for the <u> element is a black underline underneath the text. In HTML4, the <u> element was used for styling purposes. But in HTML5, the <u> element should only be used to indicate that text has non-textual annotation applied.
If you want to style a piece of text with an underline, you should use CSS instead of HTML.

The <s> Element

The strikethrough element, or <s> element for short, is used to represent text that is no longer accurate or relevant. The default styling for the <s> element is a horizontal line through the center of the text.

<s> element example 

    <p><s>Tomorrow's hike will be meeting at noon.</s></p>
    <p>Due to unforeseen weather conditions, the hike has been canceled.</p>

Tomorrow's hike will be meeting at noon.

Due to unforeseen weather conditions, the hike has been canceled.

In this example, the first sentence is crossed out because the hike was canceled due to weather reasons.

The <s> element should never be used to just to show changes to a document. More appropriate elements in that case would be the deleted text element and the inserted text element.

The <ruby> Element

The <ruby> element represents small text shown above or below the main text. It is typically used to show the pronunciation of East Asian characters.

<ruby> example 
<ruby>明日<rp>(</rp><rt>Ashita</rt><rp>)</rp></ruby>
明日(Ashita) 
<ruby>水<rp>(</rp><rt>mizu</rt><rp>)</rp></ruby>
水(mizu)

The <rp> element provides fallback parentheses for browsers that don't support ruby annotations.
The <rt> element defines the ruby text, typically used for pronunciation or translation.
While the ruby element can be used for other types of annotations, the most common use case is for East Asian typography.
