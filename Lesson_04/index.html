Mathematical Equations and Chemical Formulas

The superscript element is used to display a piece of text as a superscript(<sup>). A superscript is a symbol or letter printed above the normal line of text.

Superscript example 
<p>2<sup>2</sup>(2 squared) is 4.</p>

22 (2 squared) is 4.

Common use cases for the superscript element would include exponents, superior lettering, and ordinal numbers.

Here is an example using the superscript element for superior lettering 
<p>Monseigneur is often written as <strong>M<sup>gr</sup></strong>.</p>
Monseigneur is often written as Mgr.

Superior lettering refers to letters written in superscript, usually to indicate abbreviations. The letters g and r are wrapped inside superscript tags to illustrate the abbreviation in this example.

It is important to note that the superscript element should only be used for typographical reasons. Note: If you want to style a piece of text with a raised baseline, then you should use CSS instead of the superscript element.

To represent chemical equations inside HTML, you would use the subscript(<sub>) element. This element uses a subscript which displays a lowered baseline using smaller text.

Subscript example

CO2

<p>CO<sub>2</sub></p>
